# 🚀 Spam Email Detection System

A full-stack machine learning web application that detects spam messages using Natural Language Processing (NLP).

---

## 🔥 Features
- 🔐 Login UI
- 📊 Dashboard UI
- 🤖 Spam detection using Machine Learning (Naive Bayes)
- ⚡ Flask backend API
- 🎨 Clean and modern UI/UX

---

## 🛠 Tech Stack
- Python
- Flask
- HTML, CSS (Tailwind)
- Scikit-learn (ML)

---

```bash
python app.py
